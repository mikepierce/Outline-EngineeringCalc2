Natural Exponential

\[ \mathrm{e}^x = \exp(x) = \ln^{-1}(x) \qquad \int\limits \mathrm{e}^x\,\mathrm{d}x = \mathrm{e}^x + C \qquad \frac{\mathrm{d}}{\mathrm{d}x}\mathrm{e}^x = \mathrm{e}^x \]
  1. Calculate the derivatives of the following functions explicitly: \[ \newcommand{\ex}{{\mathrm{e}}} f(x) = \frac{1}{3}\left(\ex^x\right)^2 \qquad g(t) = \ex^{\sin(t)} \] \[\newcommand{\ex}{\mathrm{e}} h(\theta) = \tan\left(2\ex^\theta\right) \qquad j(x) = x^2\ex^{5x^3} \] \[\newcommand{\ex}{\mathrm{e}} k(x) = \frac{\ex^x}{x} + 17\mathrm{e}^2 \qquad \ell(x) = 3\ex^{2x}\sec(x) \]
  2. Calculate the following integrals explicitly: \[\newcommand{\ex}{\mathrm{e}} \int \left(\pi\ex^x + \ex\right) \,\mathrm{d}x \qquad \int x^2\ex^{5x^3} \,\mathrm{d}x \qquad \int\limits_1^2 \frac{\ex^{\frac{1}{x}}}{x^2} \,\mathrm{d}x \]
  3. (Stewart) Show that the function \(y = \mathrm {e}^x + \mathrm{e}^{-x/2}\) satisfies the differential equation \(2\ddot{y} - \dot{y}-y=0\,.\)
  4. (Stewart) For \(f(x) = \mathrm{e}^{3x}\) find a general formula for \(f^{(n)}(x)\), the \(n\)th derivative of \(f\).
  5. If you understand the lesson of the previous exercise, can you think of a function \(y = f(x)\) such that \(\dot{y} = 7y\)? Can you think of a function \(y = g(x)\) such that \(\ddot{y} = 7y\)?

  6. The Error Function

    (Stewart) The error function \(\mathrm{erf}\) is defined as \[\mathrm{erf}(x) = \frac{2}{\sqrt{\pi}}\int\limits_0^x \mathrm{e}^{-t^2} \,\mathrm{d}t \,.\]
    1. Show that \(\int_a^b = \mathrm{e}^{-t^2}\,\mathrm{d}t = \frac{1}{2}\sqrt{\pi}\big(\mathrm{erf}(b) - \mathrm{erf}(a)\big)\,.\)
    2. Show that the function \(y = \mathrm{e}^{x^2}\mathrm{erf}(x)\) satisfies the equation \(\dot y = 2xy + \frac{2}{\sqrt{\pi}}\,.\)
  7. We defined the constant \(\mathrm{e}\) as the unique number such that \[\int\limits_1^\mathrm{e} \frac{1}{x} \,\mathrm{d}x = 1.\] But what is this number explicitly? Can you write an algorithm to approximate the decimal presentation of the number \(\mathrm{e}\)? As a reference, to check your algorithm, past mathematicians have computed \[\mathrm{e} \approx 2.718281828459045235360287471352662497757247\,.\]
