Improper Integrals

  1. Evaluate the following integrals. Solutions to each of them can be found in Stewart. \[ \int\limits_{-\infty}^{0} x\mathrm{e}^x \,\mathrm{d}x \qquad \int\limits_{-\infty}^{\infty} \frac{1}{1+x^2} \,\mathrm{d}x \] \[ \int\limits_{0}^{1} \ln(x) \,\mathrm{d}x \qquad \int\limits_{0}^{3} \frac{1}{x-1} \,\mathrm{d}x \]
  2. For \(z\gt 1\), define the function \[A(z) = \int\limits_1^z x^{-p} \,\mathrm{d}x\]
    1. Show that if \(p=1\), then \(A(z) = \ln|z|\), but that otherwise, for \(p\neq 1\), we have \[ A(z) = \frac{1}{1-p}\left(z^{1-p}-1\right)\,. \]
    2. Show that if \(p \in (0,1]\), then \(\lim\limits_{z \to \infty} A(z) = \infty\).
    3. Now show that if \(p \gt 1\), then \(\lim\limits_{z \to \infty} A(z) = \frac{1}{p-1}\).
    Look out for the vocabulary term \(p\)-series later in the course, and remember this exercise when you see it.
  3. Decide if each of the following integrals is convergent or divergent. For solutions see Paul’s Online Math Notes. \[ \int\limits_{2}^{\infty} \frac{\cos^2(x)}{x^2} \,\mathrm{d}x \qquad \int\limits_{3}^{\infty} \frac{1}{x+\mathrm{e}^x} \,\mathrm{d}x \qquad \int\limits_{3}^{\infty} \frac{1}{x-\mathrm{e}^{-x}} \,\mathrm{d}x \] \[ \int\limits_{1}^{\infty} \frac{1 + 3\sin^4(2x)}{\sqrt{x}} \,\mathrm{d}x \qquad \int\limits_{2}^{\infty} \frac{1+\cos^2(x)}{\sqrt{x}\big(2-\sin^4(x)\big)} \,\mathrm{d}x \] \[ \int\limits_{1}^{\infty} \frac{\mathrm{e}^{-x}}{x} \,\mathrm{d}x \qquad \int\limits_{1}^{\infty} \mathrm{e}^{-x^2} \,\mathrm{d}x \qquad \int\limits_{1/2}^{\infty} \mathrm{e}^{-x^2} \,\mathrm{d}x \qquad \]
  4. Evaluate each of the following integrals, determining if the area described is or isn’t finite. Some of these are borrowed from Paul’s Online Math Notes, so their solutions can be found there. \[ \int\limits_{0}^{\infty} x\mathrm{e}^{-x} \,\mathrm{d}x \qquad \int\limits_{0}^{\pi/2} \frac{\cos(x)}{\sqrt{\sin(x)}} \,\mathrm{d}x \qquad \int\limits_{0}^{\pi/2} \frac{\cos(x)}{\sqrt{\sin(x)}} \,\mathrm{d}x \] \[ \int\limits_{1}^{\infty} \frac{1}{x^{1/3}} \,\mathrm{d}x \qquad \int\limits_{-5}^{1} \frac{1}{10+2z} \,\mathrm{d}z \qquad \int\limits_{0}^{4} \frac{x}{x^2-9} \,\mathrm{d}x \] \[ \int\limits_{-\infty}^{\infty} \frac{6w^3}{(w^4+1)^2} \,\mathrm{d}w \qquad \int\limits_{-1}^{1} \ln|x| \,\mathrm{d}x \qquad \int\limits_{1}^{\infty} \frac{1}{x^3} \,\mathrm{d}x \] \[ \int\limits_{-\infty}^{1} \frac{3}{1+x^2} \,\mathrm{d}x \qquad \int\limits_{-\infty}^{0} \frac{\mathrm{e}^{\frac{1}{x}}}{x^2} \,\mathrm{d}x \qquad \int\limits_{0}^{2} \frac{1}{(x-1)^4} \,\mathrm{d}x \]

Challenges

  1. Draw a picture of the region between the curves \(y = \ln(x)\) and \(y = \ln\left(x^2\right)\) in the lower half-plane. Is the area of this region finite? If it is, what’s its area? If not how do you know? Write down an integral that represents the volume of the solid formed by rotating this region about the \(y\)-axis. Is the volume of this solid finite?
  2. (Source) A solid is generated by rotating about the \(x\)-axis the region under the curve \(y = f(x)\), where \(f\) is a positive function and \(x \geq 0\). The volume generated by the part of the curve from \(x = 0\) to \(x = b\) is \(b^2\) for all \(b \gt 0\). Find the function \(f\).
