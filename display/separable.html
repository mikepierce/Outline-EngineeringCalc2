Separable Differential Equations

  1. Solve these differential equations. Notice that no initial conditions are provided, so you’re being asking for the general solution to each of these. (I'm pretty sure these are separable …)
    1. \(y\dot{y} + y^2 = 3ty\dot{y}+1\)
    2. \(y'\cot^2(x) + \tan(y) = 0\)
    3. \(xy' - 27y' = 3(y-9)\)
    4. \((ty)^2+2y^2 + \dot{y} = 0\)
    5. \(2\cos(t) = 3t^2-y'\)
    6. \(y'-6y = 4\)
    7. \(xy' - 2y' = 2(y-4)\)
  2. Solve these differential equations. Notice that no initial conditions are provided, so you’re being asking for the particular solution to each of these. (I'm pretty sure these are separable …)
    1. \(\frac{1}{t}\sec^2(y)\frac{\mathrm{d}y}{\mathrm{d}t} = 1\) where \(y(0)=1\).
    2. \(\mathrm{e}^t - yy' = 0\) where \(y(0) = 3\).
    3. \(y'' = x^2 + 3\) where \(y=-4\) and \(y'=2\) when \(x=0\).
  3. Given that \(y(3)=2\mathrm{e}\) find an explicit particular solution to the differential equation \[ \frac{y'}{x} = \mathrm{e}^{x-\ln\left(y^2\right)} \,. \]
