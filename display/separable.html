Separable ODEs

  1. Solve the differential equation \(\dot y = 6y^2x\).
  2. Solve the differential equation \(\dot y = 6y^2x\) subject to the initial condition \(y(1) = 1/9\).
  3. Solve the initial value problem: What is a formula \(y = f(x)\) if \(\displaystyle \frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{e}^x}{y}\) and \(y(0)=1\)?
  4. Find an explicit formula \(y = f(x)\) for the general solution to each of the following differential equations. \[ \dot y = x\mathrm{e}^{x^2-\ln\left(y^2\right)} \qquad x^2\dot y = \sec(y) - \dot y \] \[ \sin^2(y)\dot y = (1-\dot y)\cos^2(y) \qquad x\dot y = \sqrt{x^2 - (xy)^2} \]

Examples from Stewart

  1. Consider the differential equation \(\displaystyle \dot y = \frac{x^2}{y^2} \).
    1. Calculate the general solution to the differential equation.
    2. Calculate the particular solution to the differential equation subject to the initial condition \(y(0)=2\).
  2. Calculate a formula for the general solution to differential equation \[\dot y = \frac{6x^2}{2y + \cos(y)}\,.\] Note you can only find an implicit formula; you can’t solve for \(y\).
  3. Calculate an explicit formula \(y = f(x)\) for the general solution to differential equation \[\frac{\mathrm{d}y}{\mathrm{d}x} = x^2y\,.\]

Additional Exercises

  1. Solve these differential equations. Notice that no initial conditions are provided, so you’re being asking for the general solution to each of these. \[ y\dot{y} + y^2 = 3ty\dot{y}+1 \qquad y'\cot^2(x) + \tan(y) = 0 \] \[ xy' - 27y' = 3(y-9) \qquad (ty)^2+2y^2 + \dot{y} = 0 \] \[ 2\cos(t) = 3t^2-y' \qquad y'-6y = 4 \qquad xy' - 2y' = 2(y-4) \]
  2. Solve these differential equations. Notice that initial conditions are provided, so you’re being asking for the particular solution to each of these. \[ \frac{1}{t}\sec^2(y)\frac{\mathrm{d}y}{\mathrm{d}t} = 1 \quad\text{where}\quad y(0)=1 \] \[ \mathrm{e}^t - yy' = 0 \quad\text{where}\quad y(0) = 3 \] \[ y'' = x^2 + 3 \quad\text{where}\quad y=-4 \;\;\text{and}\;\; y'=2 \quad\text{when}\quad x=0 \]
  3. Given that \(y(3)=2\mathrm{e}\) find an explicit particular solution to the differential equation \[ \frac{y'}{x} = \mathrm{e}^{x-\ln\left(y^2\right)} \,. \]
