Natural Logarithms

\[\ln(x) = \int\limits_1^x \frac{1}{t}\,\mathrm{d}t\]

  1. (Stewart) Using this “geometric” definition of \(\ln(x)\), show that \(\ln(2)\) must be some number between ½ and ¾.
  2. (Stewart) Prove the “limit laws”: \[ \ln(xy) = \ln(x) + \ln(y) \qquad \ln\left(x^a\right) = a\ln(x) \]
  3. Calculate the following derivatives explicitly: \[ \frac{\mathrm{d}}{\mathrm{d}x} \ln\left(3x^2-x\right) \quad \frac{\mathrm{d}}{\mathrm{d}x} \ln(\tan(x)) \quad \frac{\mathrm{d}}{\mathrm{d}x} x^2\ln(x) \]
  4. Calculate the following integrals explicitly: \[ \int_0^{\pi/4} \tan\theta \,\mathrm{d}\theta \quad \int_1^\mathrm{e} \frac{\ln(x)}{x} \,\mathrm{d}x \quad \int \frac{x^2}{x^3-5} \,\mathrm{d}x \]
