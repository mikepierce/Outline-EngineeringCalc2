Inverse Functions

  1. Consider the function defined as \(f(x) = \displaystyle{\frac{x^3-2}{5}}\).
    1. Convince yourself this function is invertible.
    2. Write down a formula for \(\frac{\mathrm{d}}{\mathrm{d}x}f^{-1}(x)\) using the inverse function theorem.
  2. (Stewart) Consider the function defined as \(g(x) = 3x^3 + 4x^2 + 6x + 5\).
    1. Prove that \(g\) is one-to-one, and hence invertible. (Hint: use some of the phrases “increasing”, “decreasing”, “concave up” or “concave down”.)
    2. What is the value of \(\left(g^{-1}\right)'(5)\)?
    3. Can you write down a formula for the inverse of \(g\)?
